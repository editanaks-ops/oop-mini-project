# Homework: Topic 10 — Exceptions and Error Handling

Этот раздел репозитория содержит домашнее задание по теме
«Обработчики ошибок, генерация исключений и создание собственных типов исключений» на Python.

## Структура

- `exercise1.py` — функция `convert_to_int(value)` с обработкой ошибок преобразования строки в число.
- `exercise2.py` — функция `validate_user_input(data)` с валидацией входных данных и генерацией исключений.
- `exercise3.py` — пользовательское исключение `NegativeNumberError` и функция `check_positive_number(number)`.

## Описание заданий

### Упражнение 1 — convert_to_int

Реализована функция `convert_to_int(value)`, которая:

- пытается преобразовать переданное значение к целому числу с помощью `int()`;
- обрабатывает исключение `ValueError`, если преобразование невозможно;
- обрабатывает любое другое исключение базовым типом `Exception`;
- в блоке `finally` выводит сообщение о завершении попытки преобразования.

В конце файла добавлены тесты:

- корректная строка `"123"`;
- некорректная строка `"abc"`;
- значение другого типа `[1, 2, 3]`.

### Упражнение 2 — validate_user_input

Реализована функция `validate_user_input(data)`, которая:

- проверяет, что `data` является словарём (`dict`);
- проверяет наличие ключа `name` и корректность его значения;
- проверяет наличие ключа `age` и что возраст является положительным целым числом;
- при некорректных данных генерирует `TypeError` или `ValueError`;
- при необходимости использует `raise ... from e` для формирования цепочки исключений.

В файле приведены тесты для корректных и некорректных сценариев.

### Упражнение 3 — NegativeNumberError и check_positive_number

Создан класс пользовательского исключения `NegativeNumberError`, который:

- наследуется от `Exception`;
- принимает значение числа и сообщение об ошибке;
- переопределяет метод `__str__`, возвращающий информативное сообщение.

Функция `check_positive_number(number)`:

- проверяет, что число не отрицательное;
- при отрицательном значении выбрасывает `NegativeNumberError`;
- при положительном выводит подтверждающее сообщение.
