OOP Mini Project — Inheritance & Polymorphism Homework

Этот проект — домашнее задание по объектно-ориентированному программированию, посвящённое наследованию, полиморфизму и работе с классами.
Реализованы классы Product, Customer, Admin и ShoppingCart, а также логика покупок, подсчёта итоговой суммы, регистрации действий и вывод детальной информации с использованием полиморфных методов.
